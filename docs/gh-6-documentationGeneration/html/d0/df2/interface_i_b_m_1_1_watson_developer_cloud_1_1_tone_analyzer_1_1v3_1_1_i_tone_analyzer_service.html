Analyzes the tone of a piece of text. The message is analyzed for several tones - social, emotional, and language. For each tone, various traits are derived. For example, conscientiousness, agreeableness, and openness.

Parameters
textText that contains the content to be analyzed.
filterTonesFilter the results by a specific tone. Valid values for tones are emotion, language, and social.
sentencesFilter your response to remove the sentence level analysis. Valid values for sentences are true and false. This parameter defaults to true when it's not set, which means that a sentence level analysis is automatically provided. Change sentences=false to filter out the sentence level analysis.
Returns
