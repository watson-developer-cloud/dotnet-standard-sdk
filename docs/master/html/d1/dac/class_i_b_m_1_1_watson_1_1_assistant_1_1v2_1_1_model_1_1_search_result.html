List<SearchResultAnswer> IBM.Watson.Assistant.v2.Model.SearchResult.Answers
getset

An array specifying segments of text within the result that were identified as direct answers to the search query. Currently, only the single answer with the highest confidence (if any) is returned.

Notes:

  • Answer finding is available only if the search skill is connected to a Discovery v2 service instance.
  • Answer finding is not supported on IBM Cloud Pak for Data.
