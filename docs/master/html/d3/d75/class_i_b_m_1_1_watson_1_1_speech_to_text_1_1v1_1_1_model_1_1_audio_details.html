◆ Compression

For an archive-type resource, the format of the compressed archive:

  • zip for a **.zip** file
  • gzip for a **.tar.gz** file

Omitted for an audio-type resource. Constants for possible values can be found using AudioDetails.CompressionEnumValue

◆ Type

string IBM.Watson.SpeechToText.v1.Model.AudioDetails.Type
getset

The type of the audio resource:

  • audio for an individual audio file
  • archive for an archive (**.zip** or **.tar.gz**) file that contains audio files
  • undetermined for a resource that the service cannot validate (for example, if the user mistakenly passes a file that does not contain audio, such as a JPEG file). Constants for possible values can be found using AudioDetails.TypeEnumValue
