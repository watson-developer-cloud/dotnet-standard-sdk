The type of project.

The content_intelligence type is a Document Retrieval for Contracts project and the other type is a Custom project.

The content_mining and content_intelligence types are available with Premium plan managed deployments and installed deployments only. Constants for possible values can be found using ProjectDetails.TypeEnumValue
