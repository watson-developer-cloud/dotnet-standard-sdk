Detailed Description

The type of words to be listed from the custom language model's words resource:

  • all (the default) shows all words.
  • user shows only custom words that were added or modified by the user directly.
  • corpora shows only OOV that were extracted from corpora.
  • grammars shows only OOV words that are recognized by grammars.

For a custom model that is based on a next-generation model, only all and user apply. Both options return the same results. Words from other sources are not added to custom models that are based on next-generation models.
