Detailed Description

An expansion definition. Each object respresents one set of expandable strings. For example, you could have expansions for the word hot in one object, and expansions for the word cold in another. Follow these guidelines when you add terms:

  • Specify the terms in lowercase. Lowercase terms expand to uppercase.
  • Multiword terms are supported only in bidirectional expansions.
  • Do not specify a term that is specified in the stop words list for the collection.
