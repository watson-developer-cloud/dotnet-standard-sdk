◆ Status

string IBM.Watson.TextToSpeech.v1.Model.SpeakerPrompt.Status
getset

The status of the prompt:

  • processing: The service received the request to add the prompt and is analyzing the validity of the prompt.
  • available: The service successfully validated the prompt, which is now ready for use in a speech synthesis request.
  • failed: The service's validation of the prompt failed. The status of the prompt includes an error field that describes the reason for the failure.
